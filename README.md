# Simple python overlay example

A simple example of overlay app (Timer) and trayicon using Tkinter


## requirements
demo.py
- infi.systray (trayicon lib)

clickthrought.py
- pystray (using another trayicon lib)
- Pillow
- win32api
- win32gui
